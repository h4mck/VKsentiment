Aiohttp - воркер, принимающий запросы от пользователя и запрашивающий комментарии через VKapi (затем кладёт их в базе данных с пометкой unprorssed).

Handler - воркер, постоянно крутящийся в ожидании пачки новых unprocessed комментариев. После их получения обрабатывает и анализирует их тональность, а затем кладёт вместе с результатом обратно в базу данных.

Analyze - воркер, берущий обработанные комментарии из базы данных и рисующий на их основе графики.
